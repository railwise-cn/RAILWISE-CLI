YONSOON (甬算) si integra con VS Code, Cursor o qualunque IDE che supporti un terminale. Basta eseguire `yonsoon` nel terminale per iniziare.

---

## Utilizzo

- **Avvio rapido**: usa `Cmd+Esc` (Mac) o `Ctrl+Esc` (Windows/Linux) per aprire YONSOON (甬算) in una vista terminale divisa, oppure per mettere a fuoco una sessione esistente se e gia in esecuzione.
- **Nuova sessione**: usa `Cmd+Shift+Esc` (Mac) o `Ctrl+Shift+Esc` (Windows/Linux) per avviare una nuova sessione di YONSOON (甬算), anche se ne esiste gia una aperta. Puoi anche cliccare il pulsante YONSOON (甬算) nell'interfaccia.
- **Consapevolezza del contesto**: condividi automaticamente con YONSOON (甬算) la selezione corrente o la scheda attiva.
- **Scorciatoie per riferimenti file**: usa `Cmd+Option+K` (Mac) o `Alt+Ctrl+K` (Linux/Windows) per inserire riferimenti a file. Per esempio, `@File#L37-42`.

---

## Installazione

Per installare YONSOON (甬算) su VS Code e fork popolari come Cursor, Windsurf, VSCodium:

1. Apri VS Code
2. Apri il terminale integrato
3. Esegui `yonsoon` - l'estensione si installa automaticamente

Se invece vuoi usare il tuo IDE quando esegui `/editor` o `/export` dalla TUI, devi impostare `export EDITOR="code --wait"`. [Scopri di piu](/docs/tui/#editor-setup).

---

### Installazione manuale

Cerca **YONSOON (甬算)** nel Marketplace delle estensioni e clicca **Install**.

---

### Risoluzione dei problemi

Se l'estensione non si installa automaticamente:

- Assicurati di eseguire `yonsoon` nel terminale integrato.
- Verifica che la CLI del tuo IDE sia installata:
  - Per VS Code: comando `code`
  - Per Cursor: comando `cursor`
  - Per Windsurf: comando `windsurf`
  - Per VSCodium: comando `codium`
  - In caso contrario, esegui `Cmd+Shift+P` (Mac) o `Ctrl+Shift+P` (Windows/Linux) e cerca "Shell Command: Install 'code' command in PATH" (o l'equivalente per il tuo IDE)
- Assicurati che VS Code abbia i permessi per installare estensioni
