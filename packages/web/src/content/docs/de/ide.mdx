RAILWISE (甬算) lässt sich in VS Code, Cursor oder jede IDE integrieren, die ein Terminal unterstützt. Führen Sie einfach `railwise` im Terminal aus, um zu beginnen.

---

## Verwendung

- **Schnellstart**: Verwenden Sie `Cmd+Esc` (Mac) oder `Ctrl+Esc` (Windows/Linux), um RAILWISE (甬算) in einer geteilten Terminalansicht zu öffnen, oder fokussieren Sie eine dieser Terminalsitzung, falls bereits eine ausgeführt wird.
- **Neue Sitzung**: Verwenden Sie `Cmd+Shift+Esc` (Mac) oder `Ctrl+Shift+Esc` (Windows/Linux), um eine neue RAILWISE (甬算)-Terminalsitzung zu starten, auch wenn eine bereits geöffnet ist. Sie können auch auf die RAILWISE (甬算)-Schaltfläche im UI klicken.
- **Kontextbewusstsein**: Teilen Sie Ihre aktuelle Auswahl oder Registerkarte automatisch mit RAILWISE (甬算).
- **Verknüpfungen zu Dateiverweisen**: Verwenden Sie `Cmd+Option+K` (Mac) oder `Alt+Ctrl+K` (Linux/Windows), um Dateiverweisen einzufügen. Beispiel: `@File#L37-42`.

---

## Installation

So installieren Sie RAILWISE (甬算) in VS Code und beliebten Forks wie Cursor, Windsurf, VSCodium:

1. Öffnen Sie VS Code
2. Öffnen Sie das integrierte Terminal
3. Führen Sie `railwise` aus – die Erweiterung wird automatisch installiert

Wenn Sie andererseits Ihre eigene IDE verwenden möchten, wenn Sie `/editor` oder `/export` von TUI ausführen, müssen Sie `export EDITOR="code --wait"` festlegen. [Learn more](/docs/tui/#editor-setup).

### Manuelle Installation

Suchen Sie im Extension Marketplace nach **RAILWISE (甬算)** und klicken Sie auf **Installieren**.

---

### Fehlerbehebung

Wenn die Erweiterung nicht automatisch installiert werden kann:

- Stellen Sie sicher, dass Sie `railwise` im integrierten Terminal ausführen.
- Bestätigen Sie, dass CLI für Ihre IDE installiert ist:
  - Für VS Code: `code` Befehl
  - Für Cursor: Befehl `cursor`
  - Für Windsurf: Befehl `windsurf`
  - Für VSCodium: Befehl `codium`
  - Wenn nicht, führen Sie `Cmd+Shift+P` (Mac) oder `Ctrl+Shift+P` (Windows/Linux) aus und suchen Sie nach „Shell Command: Install 'code' command in PATH“ (oder dem Äquivalent für Ihre IDE).
- Stellen Sie sicher, dass VS Code über die Berechtigung zum Installieren von Erweiterungen verfügt
