RAILWISE s'intègre à VS Code, Cursor ou tout IDE prenant en charge un terminal. Exécutez simplement `railwise` dans le terminal pour commencer.

---

## Utilisation

- **Lancement rapide** : utilisez `Cmd+Esc` (Mac) ou `Ctrl+Esc` (Windows/Linux) pour ouvrir RAILWISE dans une vue de terminal divisée, ou focus sur une session de terminal existante si elle est déjà en cours d'exécution.
- **Nouvelle session** : utilisez `Cmd+Shift+Esc` (Mac) ou `Ctrl+Shift+Esc` (Windows/Linux) pour démarrer une nouvelle session de terminal RAILWISE, même si elle est déjà ouverte. Vous pouvez également cliquer sur le bouton RAILWISE dans l'interface utilisateur.
- **Contexte** : partagez automatiquement votre sélection ou onglet actuel avec RAILWISE.
- **Raccourcis de référence de fichier** : utilisez `Cmd+Option+K` (Mac) ou `Alt+Ctrl+K` (Linux/Windows) pour insérer des références de fichier. Par exemple, `@File#L37-42`.

---

## Installation

Pour installer RAILWISE sur VS Code et les forks populaires comme Cursor, Windsurf, VSCodium :

1. Ouvrir VS Code
2. Ouvrez le terminal intégré
3. Exécutez `railwise` - l'extension s'installe automatiquement

Si, d'un autre côté, vous souhaitez utiliser votre propre IDE lorsque vous exécutez `/editor` ou `/export` à partir du TUI, vous devrez définir `export EDITOR="code --wait"`. [En savoir plus](/docs/tui/#editor-setup).

---

### Installation manuelle

Recherchez **RAILWISE** dans Extension Marketplace et cliquez sur **Installer**.

---

### Dépannage

Si l'extension ne parvient pas à s'installer automatiquement :

- Assurez-vous que vous exécutez `railwise` dans le terminal intégré.
- Confirmez que la CLI de votre IDE est installée :
  - Pour VS Code : commande `code`
  - Pour Cursor : commande `cursor`
  - Pour Windsurf : commande `windsurf`
  - Pour VSCodium : commande `codium`
  - Sinon, exécutez `Cmd+Shift+P` (Mac) ou `Ctrl+Shift+P` (Windows/Linux) et recherchez « Shell Command: Install 'code' command in PATH » (ou l'équivalent pour votre IDE)
- Assurez-vous que VS Code est autorisé à installer des extensions
