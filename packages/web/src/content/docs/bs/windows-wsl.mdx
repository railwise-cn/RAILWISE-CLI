Iako YONSOON (甬算) moze raditi direktno na Windowsu, preporucujemo [Windows Subsystem for Linux (WSL)](https://learn.microsoft.com/en-us/windows/wsl/install) za najbolje iskustvo. WSL daje Linux okruzenje koje glatko radi sa YONSOON (甬算) funkcijama.

:::tip[Zašto WSL?]
WSL nudi bolje performanse fajl sistema, punu terminalsku podrsku i kompatibilnost s razvojnim alatima na koje se YONSOON (甬算) oslanja.
:::

---

## Postavljanje

<Steps>

1. **Instalirajte WSL**

   Ako vec niste, [instalirajte WSL](https://learn.microsoft.com/en-us/windows/wsl/install) prema zvanicnom Microsoft vodicu.

2. **Instalirajte YONSOON (甬算) u WSL-u**

   Kad je WSL spreman, otvorite WSL terminal i instalirajte YONSOON (甬算) jednom od [metoda instalacije](/docs/).

   ```bash
   curl -fsSL https://yonsoon.ai/install | bash
   ```

3. **Koristite YONSOON (甬算) iz WSL-a**

   Idite u direktorij projekta (Windows fajlovima pristupate preko `/mnt/c/`, `/mnt/d/` itd.) i pokrenite YONSOON (甬算).

   ```bash
   cd /mnt/c/Users/YourName/project
   yonsoon
   ```

</Steps>

## Desktop aplikacija + WSL Server

Ako preferirate YONSOON (甬算) Desktop aplikaciju, ali zelite da server radi u WSL-u:

1. **Pokrenite server u WSL-u** sa `--hostname 0.0.0.0` da dozvolite vanjske konekcije:

   ```bash
   yonsoon serve --hostname 0.0.0.0 --port 4096
   ```

2. **Povezite Desktop aplikaciju** na `http://localhost:4096`

:::note
Ako `localhost` ne radi u vasem setupu, povezte se preko WSL IP adrese (u WSL-u: `hostname -I`) i koristite `http://<wsl-ip>:4096`.
:::

:::caution
Kada koristite `--hostname 0.0.0.0`, postavite `YONSOON_SERVER_PASSWORD` da zastitite server.

```bash
YONSOON_SERVER_PASSWORD=your-password yonsoon serve --hostname 0.0.0.0
```

:::

## Web klijent + WSL

Za najbolje web iskustvo na Windowsu:

1. **Pokrenite `yonsoon web` u WSL terminalu** umjesto u PowerShell-u:

   ```bash
   yonsoon web --hostname 0.0.0.0
   ```

2. **Otvorite iz Windows browsera** na `http://localhost:<port>` (YONSOON (甬算) ispisuje URL)

Pokretanje `yonsoon web` iz WSL-a osigurava ispravan pristup fajl sistemu i terminalsku integraciju, a i dalje je dostupno iz Windows browsera.

## Pristup Windows fajlovima

WSL moze pristupiti svim Windows fajlovima kroz `/mnt/` direktorij:

- Disk `C:` → `/mnt/c/`
- Disk `D:` → `/mnt/d/`
- I tako dalje...

Primjer:

```bash
cd /mnt/c/Users/YourName/Documents/project
yonsoon
```

:::tip
Za najgladje iskustvo, razmislite da klonirate/kopirate repo u WSL fajl sistem (npr. pod `~/code/`) i tu pokrenete YONSOON (甬算).
:::

## Savjeti

- Drzite YONSOON (甬算) u WSL-u za projekte na Windows diskovima - pristup fajlovima je jednostavan
- Koristite VS Code [WSL ekstenziju](https://code.visualstudio.com/docs/remote/wsl) uz YONSOON (甬算) za integrisan tok rada
- Vase YONSOON (甬算) konfiguracije i sesije cuvaju se u WSL okruzenju na `~/.local/share/yonsoon/`
